作用域
发布于 2019-04-07 / 更新于 2019-04-07 / JS中作用域的使用。 / 热度 0 / 喜欢 0

作用域：

​ 作用域指的是一个变量的作用范围。

全局作用域：

​ 1、直接写在script标签中的 的JS代码，都在全局作用域。

​ 2、全局作用域在页面被打开时创建，在页面关闭时销毁。

​ 3、 在全局作用域中有一个全局对象window，它代表一个浏览器窗口，它由浏览器创建，我们可以直接使用。

​ 4、 创建的对象都会作为window对象的属性直接保存。

​ 创建的函数都会作为window对象的方法直接保存。

​ 例： var a=10;

​ console.log(a) ; //console.log(window.a);

​ 输出的结果均是：10.

函数作用域：

​ 1、调用函数时创建函数作用域，函数执行完毕，作用域销毁。

​ 2、每次调用函数作用域都会创建一个新的函数作用域，他们之间的关系相互独立。

​ 3 、在函数作用域中可以访问全局作用域的变量。

​ 例： var a=10

​ function fun(){

​ console.log(a);

​ }

​ fun();

​ 输出结果：10.

​ 4、在全局作用域中无法访问函数作用域。

​ function fun(){

​ var a=10；

​ }

​ console.log(a);

​ 输出结果： a is not defind

​ 注意： 当在函数作用域操作一个变量时，它会先在自身的作用域中寻找，如果有就直接输出，如果没有就在上 级中寻找，直到找完全局作用域，如果依旧找不到，则报错：ReferenceError.

​ 例： var a=20;

​ function fun1(){

​ var a=10;

​ function fun2(){

​ console.log(a);

​ }

​ fun2();

​ }

​ fun1();

​ 输出结果：10

​ 如果在多层函数中想直接调用创局变量中的变量，可以用window方法调用。

​ 例： var a=20;

​ function fun1(){

​ var a=10;

​ function fun2(){

​ console.log(window.a);

​ }

​ fun2();

​ }

​ fun1();

​ 输出结果：20
